Help

Navigation

  1. Global Navigation

    View Website :Enables a quick view of the website after any change has been made to its settings.

    Help : A link to this particular page for better understanding and easy usage for both frequent and infrequent users alike.

    Logout: The administrator's access is disconnected and it is returned to the index page of the Website.

  2. Local Navigation

    General :Basically a preference setting of the layout, visibility and slider image upload

    Home :Manually edit the content of the index page of the website.

    News and Events : Links to the news and events update panel of the settings.

    ...This same goes for all other local navigation tabs; self-explanatory.

Visibility

  • Page Visibility

    Any of the pages listed in the local navigation tabs can be hidden if deemed unnecessary. To hide any page un-check the checkbox opposite it's name in the list of default pages.
    Caution: Ensure that you have closely looked into the contents of the whole page and found none of them worth appearing on the Departmental website

  • Content Visibility

    By default, all pages of each visible are visible. However, due to lack/scarceness of content(s), any of the contents of the page may be set as hidden by un-checking the visibility preference box directly opposite the content name under the page name.

Contents

  • Text Entry

    The text contents of the web-pages are editable in the settings page. The space for editing has the current content preloaded into it.
    Short texts such as the name of the department in the layout pane of the general settings are edited in a small field and the inputs are restricted to 255 letters.
    Long texts (e.g HOD's corner in the home page) are made editable in a mini text editor to enable proper styling of the texts. The limit of the text that can be accommodated is almost unreachable. To apply a particular style to a portion of the texts engage the buttons in the text editor just as one would to a normal Microsoft Word document.

  • Table Entry

    To enter a table, as in case of the Departmental time-table, click on the table icon and select all the properties of the table then fill the table.

  • File Upload

    Some of the pages(if made visible) requires an upload of file which may be a document or image.
    Image upload requirements: The images formats acceptable are .png and .jpg . The maximum image size is 2MB.
    File upload requirements: The file formats are .pdf, .doc, .dox . The maximum file size is 5MB.

Multiple Item Entry

ADD: Additional contents can be added to the website if the need be by making use of the add button.
DELETE: Unwanted contents can be removed from the Site by making use of the delete button.
